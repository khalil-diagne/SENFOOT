DRIBBLEUR STORE — Chemins des photos (base efootball)
================================================================================
Date export : 2026-05-21

IMPORTANT
---------
- En base SQL, on stocke seulement le NOM du fichier (ex: 6c57d36bddb77ded.jpg).
- Les fichiers physiques sont sur le serveur, dans le dossier uploads/ du site.
- Après import du SQL, copiez tout le dossier uploads/ sur votre hébergeur.

Racine du site (exemple local XAMPP)
------------------------------------
  c:\xampp\htdocs\dribbleur\htdocs\

Dossiers images
---------------
  uploads/articles/     → photos des articles (boutique)
  uploads/avatars/      → photos de profil utilisateurs
  uploads/kyc/          → pièces d'identité vendeurs (KYC)

URL affichées dans le site (chemin web relatif)
-----------------------------------------------
  uploads/articles/NOM_FICHIER.jpg
  uploads/avatars/NOM_FICHIER.jpg
  uploads/kyc/NOM_FICHIER.jpg

Exemple URL complète en local :
  http://localhost/dribbleur/htdocs/uploads/articles/6c57d36bddb77ded.jpg

Exemple URL complète en production :
  https://VOTRE-DOMAINE.com/uploads/articles/6c57d36bddb77ded.jpg
  (adapter si le site est dans un sous-dossier)
